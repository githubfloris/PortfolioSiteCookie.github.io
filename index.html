Jungle Jam

An Unity 2D Game project. The project was created for an arcade machine located at my school, You can use a joystick and A and B buttons to interact with the game. The premise was to recreate a comptitive arcade game feel by making 2 players play against each other in a set of 3 random selected minigames.
Programmed:

  • Scoreboard
  • Minigames
  • Winner Screen
  • Healthbar

Lost Industry 2

Lost Industry 2 is a Half-life 2: Mod. It was made in the Source Engine which is a brush based engine, which made it easy for me to build environments without spending all the time modelling assests. My job on this project was to redesign levels (relight, add assets, make more appealing) and even come up with new levels.
